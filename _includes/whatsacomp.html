     Our first component to review is a Motherboard. What is a Motherboard? A motherboard is a device that ties together all of the parts that allow your device to work as expected. A simple way to imagine this as something a bit more relatable is relating a computer to our body. The motherboard would be like our central nervous system being the device that is used for a traffic system where commands come from one location and go to another. Certain parts are like highways while others are like one way streets. A motherboard starts out as what is called a Printed Circuit Board (PCB). Now I know you are wondering what a PCB is and whether we will be covering that. We currently only need to know what a PCB is at this time as the PCB subject has a capability to become much greater than familiarizing you with what’s a computer. If you would like to read more briefly on the PCB subject visit this wikipedia site at PCB’s. At any time you can return to the top of the page by clicking Return to introduction.

     On the motherboard you have slots and cabling to allow for upgrades of certain devices. We will be reviewing a desktop Personal Computer (PC) motherboard and a laptop PC motherboard only. Let's start with a desktop PC motherboard. You have either seen these and have used them or believe only gamers and techy people use these devices to enhance their experience. This is both true and false. This statement is true as in the desktop PC allows more versatility, upgradable features, and modifications that are not possible on a basic laptop, however, this is false as there are many other people and professions that use these desktop PC’s to allow for better capabilities to perform their jobs such as home designers and graphic designers.      Now that we have covered CPU's and motherboards we will be reviewing RAM next. In keeping with our body relation theme you can think of RAM as your short term memory. You can view RAM installed Here. Notice how the RAM lives in one of the slots that are available on the motherboard. You can view RAM prior to being installed Here. When we fall asleep for the evening we no longer remember the things we did which required little effort to remember. This could be said to be the same for RAM. RAM processes requests like keyboard clicks, mouse movements, and webpage recovery in today's advanced web browsers. RAM is simply used for tasks that require little effort. Now I know you're wondering what is the difference between 4 Gigabytes (GB) of RAM and 8GB of RAM. The difference is having the ability to run multiple programs at the same time and have your computer respond appropriately without windows and other computer functions freezing. While explaining RAM I will take this opportunity to explain virtual RAM. Virtual RAM is RAM that is build on your Hard Drive to compensate for the overusage of the RAM you do have in your system. Having virtual RAM could help when loading up webpages that are regularly visited, however, virtual RAM also hurts your system as system performance is degraded when using virtual RAM. How system performance is degraded will be explained in the next paragraph which will cover Hard Disk Drives. Return to introduction.
